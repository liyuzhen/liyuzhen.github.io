电子乐歌名后缀科普

Posted by eugene lee on 2018-11-08

听电音的老司机可能会发现，很多歌名会带一些奇奇怪怪的后缀，例如 Remix、VIP、Extended Mix 等等。这篇文章就给大家科普一下这些后缀代表什么意思。

Remix

Remix 不是指混音，混音是很多音乐制作过程都需要的环节，而 Remix 可以拆分为 “Re” + “mix”，可译为 “再混音”。

音乐制作人对一首歌进行 Remix，通常都这么改：

  • 添加、更换、删除一段片段；
  • 添加、修改音轨；
  • 拿到原工程文件，对多音轨进行修改；
  • 有 Drop 的歌曲，往往会替换新的 Drop（高潮部分）；
  • 重新编曲或更改整首歌的结构；
  • 修改 BPM（Beat Per Minute 每分钟节拍数）；
  • 只保留人声，其他部分全部重做
  • …

Remix 简单来讲就是：保留原版的东西，而又修改出自己的东西。

这里推荐两首歌，一首 Skrillex 的原版和一首 Zedd 的 Remix：

『Breakn’ A Sweat - Skrillex / The Doors』 歌曲发行于 2012.03.30

『Breakn’ a Sweat (Zedd Remix) - Zedd』 歌曲发行于 2013.09.24

Remixes

顾名思义，多首 Remix，多位音乐制作人对同一首原创歌曲的进行多个 “再混音” 的合集，一般以专辑形式出现。

『All Falls Down (Remixes)』 发行于 2017.12.01

VIP (VIP Mix)

全称 Variation In Production，音乐制作人对自己的作品的 Remix 就叫做 VIP。跟 Original Mix 一个意思。

『All The Way - Hellberg』 歌曲发行于 2016.08.19

『All The Way (VIP Mix) - Hellberg』 歌曲发行于 2016.12.07

Original Mix

原创混音制作，跟 VIP 同一个意思。

『XOXO (Original Mix) - Laidback Luke / Ina / Ralvero / Sirah』 歌曲发行于 2017.02.20

『XOXO (Acoustic Version) - Laidback Luke / Ina / Ralvero』 歌曲发行于 2017.02.20

Extended Mix

Extended Mix 的做法大部分都是在原版的开头和结尾处再加上一段，加个前奏和收尾，让作品表达的内容更加完整。（个人觉得，如果原版听多了再去听 Extended Mix 版本，部分加长版歌曲会让人觉得进入 Drop(高潮) 的过程比较拖沓，不够直接。例如下面推荐的 Future Bass 风格的 Rebel（俗称大哥大）。）

『Rebel (Extended Mix) - Lady Bee / Jalise Romy』 歌曲发行于 2017.03.31

『Rebel - Lady Bee / Jalise Romy』 歌曲发行于 2017.03.31

Bootleg

音标 [ˈbutˌlɛɡ]，翻译为 “盗版”，其实就是盗版。没有版权的 Remix 都叫做 Bootleg。

推荐一首 Deep Down Low Bootleg，专辑封面很酷。

『Deep Down Low (Tea Bootleg) - Tea』

Mashup

Mashup 准确点来说是属于音乐风格，不属于制作音乐的方式。不同歌曲的 Mashup 音乐风格也有所不同，Mashup 不单单指一种音乐风格。mashup 自身存在的同时，不单属于 mashup 这个音乐风格，还属于其他音乐风格。主要制作方式是将多首成品音乐、已发行的音乐的音轨进行重叠、拼接等 (Remix) 操作，将其混在一起，形成一首新的音乐。（不同于歌曲串烧）

Mashup 就是一种别致的 Remix，但它自称一派。Mashup 又可以叫做 Bootleg，因为 Mashup 基本都是没有版权的。

推荐一首 Mashup，将 R3hab 与 KSHMR 合作的 『Strong』 和 Sick Individuals 与 DBSTF 合作的『Into the Light』 进行融合。

『Into The Strong Light (Wavers Mashup) - Wavers』 歌曲发行于 2016.04.06

Edit

Edit 就是剪辑，对原曲进行少量的修改 (minor change to the song)，通常会对歌曲的 Drop、拍子等方面进行调整。调整的力度、幅度都是远远不及 Remix 的。

下面就推荐一首大菠萝的 1994 Edit.

『1994 (Don Diablo Edit) - Don Diablo / L’Tric / Miles Graham』 歌曲发行于 2016.04.08

『1994 - L’Tric』 歌曲发行于 2016.04.08

Edit 还会衍生出 Radio Edit (电台录制后后期进行剪辑)、Vocals Edit (Vocal 人声部分剪辑) 等各种形式。

feat.

featuring，简称 ft.。这个标记就非常常见了，流行音乐也有，标记后面跟的是歌手的名字，代表这首歌有某某歌手参与。而在电音中，用此形式来注明人声部分的歌手名。

推荐一首电子乐，最近抖音上也挺火的，火到快毁了这首歌了，非常尴尬。

『Nevada (feat. Cozi Zuehlsdorff) - Vicetone / Cozi Zuehlsdorff』 歌曲发行于 2016.06.24

Vocal Mix
Vocal 就是电音中的人声部分，混音版本叫 Vocal Mix，用别人的原创音乐作为伴奏并嵌入自己准备的人声样本而形成的另一种版本单曲。

Vocal Version
有些歌会专门附带一个 “Vocal Version”，表示一首电音被作者添加了人声后的版本。与 Vocal Mix 的区别是，Vocal Version 是作者本人添加人声后的版本。

Vocals Mashup

Restrung
不常用，旋律优美的名作才有可能发布这个版本，Restrung 是一首原创歌曲的 “原生乐器版本”，更确切的理解是把“弦乐部分”从原版中抽离出来，因此鼓点是几乎听不到的，更别说合成音色。

Mixtape
混录集，最初用于说唱界，是一种重要的音乐宣传方式，通常只有非常成功的歌手/制作人才会出版一个 Mixtape，即个人杰出作品锦集。现在，也运用到电子乐中。

XXXX Cover
代表这首歌的人声部分是由其他歌手翻唱的，新的 Vocal 代替的原版的 Vocal，通常不需要电音制作人参与，当翻唱一首歌，要带上 Cover，不然属于侵权。

XXXX EP
一种由两三首歌或者是一首歌的两三个版本组成的专辑，以 EP 发布，迷你专辑，最常见形式一首原创歌曲（Original Mix）和一首扩展混音（Extended Mix）

MIX: various tracks (themes, songs) one behind the other which each ending of the track is blended with the start from next one track, so it seems like one long long and only track (usually a mix last at least one hour or two)
